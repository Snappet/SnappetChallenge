Start

The reports that gives a teacher an overview of how and what his class has been working on on Tuesday 2015-03-24 11:30:00 UTC can be seen on index.html.

Development

  • This webpage was developed as a Visual Studio 2017 folder but is IDE agnostic;
  • All the dependencies are retrieved from CDN's and the datasource is fetched from GitHub: an internet connection is thus required;
  • The main used library is PivotTable.js which can use D3.js and C3.js renderers.
