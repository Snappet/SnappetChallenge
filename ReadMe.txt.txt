Omschrijving
Zipfile bevat de Snappet Challenge .Net solution van Ben d'Hont.

Vereisten
Het project is gemaakt in .Net Core MVC en kan geopend/gestart worden met:
 - Visual Studio 2015 update 3 + .net Core 1.0
 - Visual Studio 2017 (RC)
	
Applicatie
De applicatie bevat 2 functies die direct de starten zijn vanaf de homepagina in het menu bovenin de pagina.

 - "Resultaten per dag": Is een pagina die alle antwoorden voor een bepaalde dag groepeerd per Subject, Domain en LearningObjectice. Als eerste kies je een datum en vervolgens worden de resultaten geladen. Naas het totaal aantal anwoorden kun je er ook het aantal goede (groen) en foute (rood) antwoorden zien van die dag. Overzicht is in te klappen d.m.v. het klikken op de titel.
 - "Antwoorden per week": Geeft het aantal antwoorden per week weer gegroepeerd per Subject.
