# Snappet Challenge

This is my contribution to the Snappet Challenge.

## Prerequisites
- Visual Studio 2013 
- NuGet

## How to run
- Open the .sln file in Visual Studio
- Run the project in IIS Express

The NuGet packages should be downloaded and installed automatically.
